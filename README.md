# Random-gif-generator
The Random GIF Generator is a fun and interactive web application designed to provide users with entertaining GIFs instantly. The website is divided into two main sections for a better user experience.

The first section displays random GIFs, where users can generate a new GIF with a single click. This feature uses an external GIF API to fetch and display trending or completely random GIFs, making the experience dynamic and unpredictable.

The second section allows users to search and generate GIFs based on their input. Users can enter a keyword or phrase into an input field, and the application fetches relevant GIFs matching the query. This provides a more personalized experience, enabling users to find GIFs related to specific emotions, topics, or trends.
