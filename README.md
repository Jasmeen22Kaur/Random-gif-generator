# Random-gif-generator
The Random GIF Generator is a fun and interactive web application designed to provide users with entertaining GIFs instantly. The website is divided into two main sections for a better user experience.

The first section displays random GIFs, where users can generate random GIF by clicking on 'Generate'.This feature uses an external GIF API to fetch and display trending or completely random GIFs, making the experience dynamic and unpredictable.
The second section allows users to search and generate GIFs based on their input.

I have used useGif.js hook for getting random or input based GIFs, which reduces code redundancy and results in fast execution of the website.
